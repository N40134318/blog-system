本阶段核心目标：

Switch）完整闭环**

------------------------------------------------------------------------

## 🚀 二、核心成果

### 1️⃣ 迁移系统（已实测通过）

#### 打包脚本

`scripts/package-migration.sh`

能力： - 导出项目配置（deploy / scripts / docs） - 导出 MySQL
数据（mysqldump） - 导出 uploads - 导出 nginx 配置 - 导出 docker
运行状态

------------------------------------------------------------------------

#### 恢复脚本

`scripts/restore-migration.sh`

能力： - 解压迁移包 - 恢复项目文件 - 恢复 uploads - 启动 MySQL / Redis -
重建数据库 - 导入 SQL - 启动完整服务 - 健康检查

------------------------------------------------------------------------

### 2️⃣ 环境切换能力

`scripts/switch-env.sh`

能力： - dev / prod 切换 - 自动重写 nginx 配置 - reload nginx -
无需手动改配置

------------------------------------------------------------------------

### 3️⃣ 健康检查

`scripts/health-check.sh`

能力： - Backend 检查 - Frontend 检查 - 一键验证环境状态

------------------------------------------------------------------------

### 4️⃣ Nginx 自动化

`scripts/write-nginx-env.sh`

能力： - 通过脚本生成 nginx 配置 - 避免手动格式错误 - 支持 dev / prod
切换

------------------------------------------------------------------------

## 🎯 五、关键设计原则

### 1️⃣ 数据与容器解耦

-   使用 SQL dump
-   不直接复制 volume

------------------------------------------------------------------------

### 2️⃣ 可重复恢复

-   任意机器可恢复
-   不依赖环境状态

------------------------------------------------------------------------

### 3️⃣ 环境隔离

-   dev / prod 独立
-   互不影响

------------------------------------------------------------------------

### 4️⃣ 自动化优先

-   nginx 自动生成
-   切换脚本化
-   健康检查脚本化

------------------------------------------------------------------------

## ⚠️ 六、踩坑总结（关键经验）

### 1. nginx 502

原因： - upstream 指向错误 - 端口不一致

------------------------------------------------------------------------

### 2. SQL 未导入

表现： - 页面空数据

------------------------------------------------------------------------

### 3. Flyway 校验失败

原因： - schema 与 migration 不一致

------------------------------------------------------------------------

### 4. Docker 网络问题

表现： - connect refused

------------------------------------------------------------------------

### 5. 权限问题

表现： - uploads 不可访问

------------------------------------------------------------------------
